## Cara Menjalankan

1. Clone Repo
``` 
git clone https://github.com/ayusekar94/gits-msib4-uts
```

2. Copy env dari env.example
3. Generate key
``` 
php artisan key:generate
```

4. Mingrate tabel
``` 
php artisan migrate:fresh --seed
```

5. Generate link
``` 
php artisan storage:link
```

6. Run
``` 
php artisan serve
```
